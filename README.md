# Seriktes Robot

Seriktes — это автономный робот-ассистент для розничных магазинов.  
Он умеет общаться с людьми через живой AI-аватар, распознавать команды,  
двигаться по пространству, избегать препятствий и взаимодействовать с окружающей средой с помощью роботизированной руки.

---

## Возможности робота

• Реальное общение с клиентами через AI-аватар  
• Голосовое взаимодействие: распознаёт речь, отвечает голосом  
• Активация по горячему слову: «Сериктес!»  
• Понимание команд, например:
  – «подними руку»  
  – «опусти руку»  
  – «отъедь назад»  
  – «вперёд»  
  – «стоп»  
• Обнаружение объектов и препятствий с помощью компьютерного зрения  
• Автоматическое остановление при обращении пользователя  
• Автоматическое объезд препятствий  
• Манипуляция объектами и взаимодействие через роботизированную руку  
• Мониторинг состояния системы (нагрузка, частота работы, карта окружения)

## Описание работы робота

Робот постоянно анализирует окружающую среду с помощью камеры и компьютерного зрения.  
Если он видит препятствие, он останавливается и при необходимости использует манипулятор,  
чтобы убрать объект.

Когда пользователь говорит «Сериктес!», робот переключается в режим голосового управления,  
останавливает движение и ждёт команд. После получения команды робот выполняет действие  
и использует AI-аватар, чтобы сообщить об успешном выполнении.

AI-аватар в реальном времени озвучивает ответы и отображает эмоции, создавая естественное  
взаимодействие с клиентами.

## Интеграция и использование

Проект построен модульно, поэтому каждый компонент может использоваться отдельно или в связке:  
навигация, манипулятор, системы восприятия, голосовой интерфейс и аватар могут быть как  
автономными модулями, так и единым комплексом, который формирует поведение Seriktes.

---

## Назначение проекта

Seriktes создавался как робот-ассистент для магазинов и торговых центров.  
Он способен помогать клиентам, отвечать на вопросы, взаимодействовать с объектами  
и демонстрировать поведение интеллектуальной системы.

Проект подходит для демонстраций, хакатонов, прототипирования и исследований в  
области робототехники и искусственного интеллекта.
